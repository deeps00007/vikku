Privacy Policy

Last Updated: May 2026

VoiceKhaata respects your privacy and is committed to protecting your personal information. This Privacy Policy explains how our application collects, uses, and protects user data.

1. Information We Collect

VoiceKhaata may collect the following information:

  • Voice input provided by the user
  • Speech-to-text converted ledger entries
  • Basic authentication information
  • Device-related information required for app functionality

2. Microphone Permission

VoiceKhaata uses the device microphone permission (RECORD_AUDIO) to allow users to create voice-based ledger and expense entries.

Audio is accessed only when the user actively uses voice input features inside the app.

3. How We Use Information

The collected information is used to:

  • Provide voice-based ledger functionality
  • Convert speech into text entries
  • Improve app performance and user experience
  • Maintain authentication and app security

4. Data Storage

User data may be securely stored using cloud services such as Firebase or secure backend systems to provide synchronization and app functionality.

5. Data Sharing

VoiceKhaata does not sell or rent user data to third parties.

Data may only be shared with trusted service providers required for app functionality, such as authentication, cloud storage, analytics, or notification services.

6. Security

We implement reasonable security measures to protect user information from unauthorized access, disclosure, or misuse.

7. Children's Privacy

VoiceKhaata is not intended for children under the age of 13.
